# stm8s-1838-analyzer
## Features
* Analyse infrared remote controller
* Record pulse width

## Wires
```
  1838       STM8S003F3P3     USBtoUART
+------+     +----------+     +-------+
|      |     |          |     |       |
|  Data|<--->|A2      TX|<--->|RX     |
|      |     |          |     |       |
|   GND|<--->|GND    GND|<--->|GND    |
|      |     |          |     |       |
+------+     +----------+     +-------+
```
